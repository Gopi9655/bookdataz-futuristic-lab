# Design Contract

This contract defines the required visual system for future redesign work. It does not authorize content or data changes.

## Layout

| Element | Required value |
| --- | --- |
| Maximum width | `1280px` |
| Section wrapper | `mx-auto max-w-7xl` |
| Mobile horizontal padding | `px-5` |
| Tablet horizontal padding | `px-8` |
| Desktop horizontal padding | `px-10` |
| Section vertical padding | `py-20 lg:py-28` |
| Card gap | `gap-6 lg:gap-8` |
| Card padding | `p-6 lg:p-8` |
| Card radius | `rounded-3xl` |
| Button radius | `rounded-full` |

## Brand Style

- Use dark navy as the primary foundation.
- Use orange for calls to action.
- Use blue for data accents.
- Maintain a modern B2B data intelligence aesthetic.

## Animation

- Use subtle Framer Motion animation only.
- Avoid animation clutter.
- Do not introduce layout-breaking floating elements.
