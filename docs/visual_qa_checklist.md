# Visual QA Checklist

Complete this checklist after approved visual changes.

- [ ] Desktop alignment is consistent.
- [ ] Mobile alignment is consistent.
- [ ] Navbar dropdown works correctly.
- [ ] Hero spacing follows the design contract.
- [ ] Cards are visually consistent.
- [ ] Footer is consistent across routes.
- [ ] No horizontal or unintended vertical overflow.
- [ ] No empty image `src` values.
- [ ] No undefined Tailwind classes.
- [ ] No placeholder content.
- [ ] No duplicated source-of-truth data.
- [ ] No hydration errors.
- [ ] Build result is recorded.
