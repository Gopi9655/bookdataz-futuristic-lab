# BookDataZ Futuristic Design Lab

This repo is an experimental design lab copied from the stable BookDataZ redesign branch.

Goal:
Create a very modern, futuristic, premium B2B data/database website while preserving all existing BookDataZ content and data.

This is not a cyber-security website.
This is not a hacker/terminal product.
This is not a generic AI SaaS landing page.

Brand direction:
- deep navy / dark blue
- soft off-white / white surfaces
- controlled bright blue highlights
- orange CTA/accent colour from the logo
- futuristic but corporate
- premium B2B database/data-intelligence feel

Visual target:
- 3D-style floating database cards
- layered dashboard panels
- motion blur accents
- animated data streams
- floating category chips
- selective glassmorphism
- orbital/network/data-grid visuals
- modern animated icons
- premium shadows and depth
- production-grade responsive layout

Hard rules:
- Do not change src/resource/data.js
- Do not change src/resource/mockdata.js
- Do not change category names
- Do not change counts
- Do not change subcategories
- Do not change industry slugs
- Do not change contact details
- Do not change form fields
- Do not change API behavior
- Do not invent fake metrics, testimonials, clients, or claims

Homepage order:
1. Hero section first
2. Available Data Coverage section immediately second

Available Data Coverage must preserve:
- all 18 categories
- all 180 subcategory listings
- exact counts
- real BookDataZ data only

If using Three.js:
- use it only where production-safe
- lazy-load if possible
- provide a static fallback
- respect reduced motion
- keep mobile performance strong
